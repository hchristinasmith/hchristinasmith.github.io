Classes vs. IDs: Best Practices

In HTML and CSS, classes and IDs are both used to target and style elements, but they have different purposes and rules.

Classes (class=):

  • Used to style multiple elements
  • Can apply the same class to multiple elements
  • In CSS, it's writen with a dot (.)

ID's (id=):

  • Used to uniquely identify a single element.
  • Only use an ID once per page
  • In CSS, it's written with a hash (#)

Key Differences:

Feature Class ID
Prefix . (dot) # (hash)
Uniqueness Reusable Must be unique
Specificty Lower Higher
Usage count Multiple Elements Only one Element
